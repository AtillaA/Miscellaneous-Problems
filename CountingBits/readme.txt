Counting Bits
--------------
Problem Number: #338
Difficulty: Easy


Given an integer n, determine the number of 1-bits in its binary representation, and the position of each 1-bit listed in ascending order.

Constraints:
------------
1 < n < 10^9

________________________________________________________________
----------------------------------------------------------------

Example 1:
----------
Input: n = 37
Output: [3, 1, 4, 6]
Explanation: 37 is 100101 in binary.

----------------------------------------------------------------

Example 2:
----------
Input: n = 161
Output: [3, 1, 3, 8]
Explanation: 161 is 10100001 in binary.
